## 접근
부터 쉽지가 않다. 다른 사람의 풀이를 참고하여 천천히 접근하기로 했다....

순서
1. 테이블 정의하기
2. 점화식 찾기
3. 초기값 정하기

#### 1. 테이블 정의하기
dp[i]=i를 1로 만들 때 연산해야하는 횟수 최솟값

#### 2. 점화식 찾기
dp[12]를 예로 들면
- 3으로 나누기

    dp[12]=dp[4]+1<br>
    왜?: 12/3=4, 4에 도달하는 최소 연산 횟수에 +1을 하면 12에 도달
- 2로 나누기

    dp[12]=dp[6]+1<br>
    마찬가지. 12/2=6, 6에 도달하는 최소 연산 횟수에 +1을 하면 12에 도달
- 1 빼기

    dp[12]=dp[11]+1<br>
    12-1=11, 11에 도달하는 최소 연산 횟수에 +1을 하면 12에 도달

이를 정리하면 다음과 같다.
- 3으로 나누기

    dp[i]=dp[i/3]+1
- 2로 나누기

    dp[i]=dp[i/2]+1
	
- 1 빼기
    dp[i]=dp[i-1]+1

#### 3. 초기값 정하기
dp[0]=dp[1]=0<br>
왜?: 이미 1이니까

이를 바탕으로 코드를 짠다.

```java
import java.util.*;

public class Main {

	public static void main(String args[]) {
		Scanner sc=new Scanner(System.in);
		int n=sc.nextInt();
		int[] dp=new int[n+1];
		dp[1]=0;

		//1은 필요 없기 때문에 2부터 시작
		for(int i=2;i<=n;i++){
			//1을 빼는 경우(최대값)
			dp[i]=dp[i-1]+1;
			//2로 나누는 경우
			//1을 빼는 경우와 비교하여 더 작은 걸 선택
			if(i%2==0) dp[i]=Math.min(dp[i],dp[i/2]+1);
			//3으로 나누는 경우
			//1빼는 경우와 비교하여 더 작은 걸 선택
			if(i%3==0) dp[i]=Math.min(dp[i],dp[i/3]+1);
		}
		System.out.println(dp[n]);
	}
}
```
